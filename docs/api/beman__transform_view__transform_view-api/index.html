class beman::transform_view::transform_view

class beman::transform_view::transform_view
  : public std::ranges::view_interface< transform_view< V, F > >

An updated [transform_view](#classbeman_1_1transform__view_1_1transform__view) whose iterator constructs an F on the fly rather than using the one stored in the view when F can be trivially constructed and destructed. This makes this [transform_view](#classbeman_1_1transform__view_1_1transform__view) conditionally borrowable. Note that this template derives from [view.interface], and so has many operations not explicitly documented below.
